## 📊 Project Overview

This project analyzes gym member behavior, workout patterns, and attendance trends using **SQL** for data extraction and **Tableau** for interactive visualization. The dashboard provides actionable insights for gym management to optimize staffing, reduce no-shows, and improve member retention.

---

## 🛠️ Technologies Used
- **SQL** (MySQL) - Data exploration and analysis
- **Tableau** - Interactive dashboard creation
- **Excel/CSV** - Data cleaning
