## 📊 Project Overview

This project analyzes gym member behavior, workout patterns, and attendance trends using **SQL** for data extraction and **Tableau** for interactive visualization. The dashboard provides actionable insights for gym management to optimize staffing, reduce no-shows, and improve member retention.

---

## 🛠️ Technologies Used
- **SQL** (MySQL) - Data exploration and analysis
- **Tableau** - Interactive dashboard creation
- **Excel/CSV** - Data cleaning

---

---

## 📋 Dashboard Features

### 1. Peak Hour Visit Analysis
- Hourly traffic distribution (5 AM - 10 PM)
- Identify busiest times for staffing

### 2. Total Visits by Day of Week
- Monday through Sunday breakdown
- Weekend vs weekday patterns

### 3. Top Workout Types by Membership
- Cardio | CrossFit | HIIT | Strength | Yoga
- Filterable by membership type

### 4. Factors Affecting Workout Duration
- Duration comparison across workout types
- Seasonal patterns (Spring/Summer/Fall)

### 5. No-Show Patterns by Month
- January through September trends
- Identify high-risk months

### 6. Workout Preferences by Age Group & Gender
- Young Athletes (<25)
- Peak Performance (25-34)
- Maintenance (35-44)
- Active Aging (45-54)
- Health Focus (55-64)
- Senior Fitness (65+)

---

## 🔍 Key Insights Summary

| # | Insight | Business Impact |
|---|---------|-----------------|
| 1 | **52% no-show rate** costs revenue | Implement reminder system |
| 2 | Peak hours = 5-9 AM & 5-8 PM | Staff accordingly |
| 3 | HIIT most popular across all ages | Schedule daily HIIT classes |
| 4 | Cardio preferred by monthly members | Target for upgrades |
| 5 | Quarterly members = highest no-shows (53.9%) | Charge no-show fees |
